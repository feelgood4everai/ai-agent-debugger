# AI Agent Debugger

Visual debugging tool for multi-agent AI systems with step-by-step tracing.

## What it does

This demo shows how an AI Agent Debugger can trace execution through multiple agents:
- **Intake Agent**: Parses the incoming query
- **Validator Agent**: Checks if the query is valid  
- **Processor Agent**: Performs the main processing
- **Responder Agent**: Generates the final response

## How to use

1. Enter a query in the text box (e.g., "Process order #12345")
2. Click "üîç Trace Execution"
3. See the step-by-step execution trace
4. View agent metrics (calls, latency, status)
